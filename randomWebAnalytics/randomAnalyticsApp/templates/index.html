About the website:

Random Analytics performs an analysis of different datasets from various sources.

The aim of this application is to apply various aspects of Data Science like data mining, model building, predictive modelling and visualization for data from real world applications.

One of the key focus is to become familiar with d3.js, which is a javascript library for interactive web applications.
